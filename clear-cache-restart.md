# キャッシュクリアと開発サーバー再起動手順

## 手順

### Step 1: 開発サーバーを完全に停止

1. ターミナルで開発サーバーが起動しているウィンドウを確認
2. **Ctrl + C** を2回押して完全に停止
3. ターミナルにプロンプト（`C:\Users\...>`）が表示されるまで待つ

### Step 2: .nextフォルダを削除（ファイルエクスプローラーで）

1. ファイルエクスプローラーを開く
2. 以下のパスに移動:
   ```
   C:\Users\taka-\OneDrive\デスクトップ\project\survey-app
   ```
3. `.next`フォルダを探す
   - 見つからない場合は、エクスプローラーの「表示」タブ → 「非表示項目」にチェックを入れる
4. `.next`フォルダを右クリック → 「削除」
   - もし削除できない場合は、フォルダ名を `.next_old` にリネーム

### Step 3: 開発サーバーを再起動

コマンドプロンプトで以下を実行:

```cmd
cd C:\Users\taka-\OneDrive\デスクトップ\project\survey-app
npm run dev
```

### Step 4: ブラウザでハードリロード

1. ブラウザで `http://localhost:3000/survey` にアクセス
2. **Ctrl + Shift + R** または **Ctrl + F5** を押してハードリロード
   - これでブラウザのキャッシュもクリアされます
