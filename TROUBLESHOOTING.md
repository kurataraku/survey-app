# 接続エラー解決ガイド

## 問題: "localhost で接続が拒否されました"

### 解決方法1: サーバーを再起動

1. **現在実行中のサーバーを停止**
   - ターミナルで `Ctrl + C` を押す

2. **再度起動**
   ```bash
   npm run dev
   ```

3. **完全に起動するまで待つ**
   - `✓ Ready in X.Xs` というメッセージが表示されるまで待つ

### 解決方法2: ポートが使用されている場合

別のアプリケーションがポート3000を使用している可能性があります。

1. **別のポートで起動**
   ```bash
   npm run dev -- -p 3001
   ```
   その後、ブラウザで `http://localhost:3001/survey` を開く

2. **または、package.jsonを編集**
   ```json
   "scripts": {
     "dev": "next dev -p 3001"
   }
   ```

### 解決方法3: ファイアウォールの確認

Windowsファイアウォールがブロックしている可能性があります。

1. Windowsの設定 → プライバシーとセキュリティ → Windowsセキュリティ
2. ファイアウォールとネットワーク保護
3. アプリがファイアウォールを通過できるようにする
4. Node.js を許可する

### 解決方法4: ブラウザのキャッシュをクリア

1. ブラウザで `Ctrl + Shift + Delete` を押す
2. キャッシュされた画像とファイルを削除
3. 再度 `http://localhost:3000/survey` を開く

### 解決方法5: 127.0.0.1を使用

`localhost` の代わりに `127.0.0.1` を試す：

```
http://127.0.0.1:3000/survey
```

## 確認事項

- [ ] サーバーが完全に起動している（`✓ Ready` が表示されている）
- [ ] ターミナルにエラーメッセージが表示されていない
- [ ] ポート3000が使用可能である
- [ ] ブラウザで正しいURLを開いている
