# 開発サーバーの起動手順（PowerShellエラー回避版）

## 手順

### Step 1: VS Codeでターミナルを開く

1. VS Codeの上部メニューから: `ターミナル` → `新しいターミナル`
   - または、キーボードショートカット: **Ctrl + Shift + `**（バッククォート）

### Step 2: ターミナルをコマンドプロンプトに切り替える

1. ターミナルウィンドウの右上を見る
2. `+` ボタンの横にある **`▼`**（下向き矢印）をクリック
3. メニューから **`コマンド プロンプト`** または **`Command Prompt`** を選択

### Step 3: 開発サーバーを起動

コマンドプロンプトが開いたら、以下のコマンドを実行:

```cmd
npm run dev
```

### Step 4: 起動の確認

以下のようなメッセージが表示されれば成功です:

```
▲ Next.js 16.1.1
- Local:        http://localhost:3000
- Ready in 2.3s
```

### Step 5: ブラウザで確認

メッセージが表示されたら、ブラウザで以下にアクセス:

```
http://localhost:3000
```

または、アンケートフォームに直接アクセス:

```
http://localhost:3000/survey
```

---

## トラブルシューティング

### まだエラーが出る場合

代わりに、以下のコマンドを試してください:

```cmd
npx next dev
```

### ポート3000が使用中の場合

別のポートで起動:

```cmd
npx next dev -p 3001
```

この場合、`http://localhost:3001` にアクセスしてください。

---

## 開発サーバーの停止方法

開発サーバーを停止するには、ターミナルで **Ctrl + C** を押してください。
