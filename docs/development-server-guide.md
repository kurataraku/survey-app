# 開発サーバー起動ガイド

## 開発サーバーが起動しているかの確認方法

### 方法1: ブラウザで確認（最も簡単）

1. ブラウザを開く
2. 以下のURLにアクセス:
   ```
   http://localhost:3000
   ```
3. ページが表示されれば、サーバーは起動しています
4. 「サイトにアクセスできません」や「接続が拒否されました」と表示されれば、サーバーは起動していません

### 方法2: ターミナル/コマンドプロンプトで確認

ターミナル（コマンドプロンプト）で以下のコマンドを実行:

```bash
curl http://localhost:3000
```

または、PowerShellの場合:

```powershell
Invoke-WebRequest -Uri http://localhost:3000
```

エラーが表示されなければ、サーバーは起動しています。

### 方法3: ターミナルのウィンドウを確認

`npm run dev`コマンドを実行したターミナルのウィンドウを確認:

- **起動している場合**: ターミナルに以下のようなメッセージが表示されています:
  ```
  ▲ Next.js 16.x.x
  - Local:        http://localhost:3000
  - Ready in X.Xs
  ```

- **起動していない場合**: そのようなメッセージは表示されていません

## 開発サーバーの起動方法

### Step 1: プロジェクトフォルダに移動

ターミナル（コマンドプロンプト）を開き、プロジェクトフォルダに移動:

```bash
cd C:\Users\taka-\OneDrive\デスクトップ\project\survey-app
```

### Step 2: 開発サーバーを起動

以下のコマンドを実行:

```bash
npm run dev
```

### Step 3: 起動の確認

ターミナルに以下のようなメッセージが表示されれば成功:

```
▲ Next.js 16.x.x
- Local:        http://localhost:3000
- Ready in X.Xs
```

このメッセージが表示されたら、ブラウザで `http://localhost:3000` にアクセスできます。

## 開発サーバーの停止方法

開発サーバーを停止するには、ターミナルで **Ctrl + C** を押してください。

---

## トラブルシューティング

### エラー: "EADDRINUSE: address already in use :::3000"

ポート3000が既に使用されている場合のエラーです。

**解決方法**:
1. 既に起動している開発サーバーを停止（Ctrl + C）
2. または、別のポートで起動:
   ```bash
   npm run dev -- -p 3001
   ```
   この場合、`http://localhost:3001` にアクセスしてください

### エラー: "Cannot find module"

依存関係がインストールされていない可能性があります。

**解決方法**:
```bash
npm install
```

その後、再度 `npm run dev` を実行してください。
