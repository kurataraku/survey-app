# Epic2: .nextフォルダの手動削除手順

## 重要な解決方法

ロックファイルの問題が続く場合、**エクスプローラーから手動で削除**するのが最も確実です。

## 手順

### Step 1: すべてのNodeプロセスを停止

**タスクマネージャーを使用：**

1. `Ctrl + Shift + Esc`でタスクマネージャーを開く
2. 「プロセス」タブを開く
3. 「Node.js」または「node」という名前のプロセスをすべて見つける
4. 各プロセスを右クリック → 「タスクの終了」
5. すべてのNodeプロセスを終了するまで繰り返す

### Step 2: エクスプローラーで.nextフォルダを削除

1. **エクスプローラーを開く**
   - `Windowsキー + E`を押す

2. **プロジェクトフォルダに移動**
   ```
   C:\Users\taka-\OneDrive\デスクトップ\project\survey-app
   ```

3. **.nextフォルダを表示**
   - 「表示」タブをクリック
   - 「隠しファイル」にチェックを入れる
   - `.next`フォルダが表示される

4. **.nextフォルダを削除**
   - `.next`フォルダを右クリック
   - 「削除」を選択
   - 確認ダイアログで「はい」をクリック

5. **削除できない場合**
   - エクスプローラーを閉じて再度開く
   - PCを再起動（最後の手段）

### Step 3: 開発サーバーを起動

1. Cursorのターミナルで：
   ```powershell
   cd "C:\Users\taka-\OneDrive\デスクトップ\project\survey-app"
   npm run dev
   ```

2. 「✓ Ready」が表示されるまで待つ

## なぜ手動削除が効果的か

- PowerShellコマンドでは削除できない場合がある（ファイルロック、権限の問題など）
- エクスプローラーから削除すると、より確実に削除できる
- 削除できない場合、エラーメッセージが明確に表示される
