# キャッシュクリアの簡単な方法

## 方法1: VS Codeのエクスプローラーから削除（推奨）

### Step 1: 開発サーバーを停止

1. ターミナルで開発サーバーが起動しているウィンドウを確認
2. **Ctrl + C** を押して停止
3. ターミナルにプロンプト（`C:\Users\...>`）が表示されるまで待つ

### Step 2: VS Codeで.nextフォルダを表示

1. **VS Codeの左側のエクスプローラー（ファイル一覧）を確認**
2. もし`.next`フォルダが見えない場合：
   - エクスプローラーの上部の「**…**」（3つの点）をクリック
   - 「**表示**」→「**隠しファイルを表示**」をクリック
   - または、エクスプローラーの設定アイコンをクリックして「**Show Hidden Files**」にチェック

### Step 3: .nextフォルダを削除

1. `.next`フォルダを見つける
2. `.next`フォルダを**右クリック**
3. 「**削除**」をクリック
4. 確認ダイアログで「**削除**」をクリック

### Step 4: 開発サーバーを再起動

コマンドプロンプトで以下を実行:

```cmd
npm run dev
```

### Step 5: ブラウザでハードリロード

1. ブラウザで `http://localhost:3000/survey` にアクセス
2. **Ctrl + Shift + R** または **Ctrl + F5** を押してハードリロード

---

## 方法2: .nextフォルダが見つからない場合

`.next`フォルダが存在しない場合、開発サーバーがまだ一度もビルドされていない可能性があります。その場合は、以下の手順で進めてください：

### Step 1: 開発サーバーを停止

1. ターミナルで **Ctrl + C** を押して停止

### Step 2: 開発サーバーを再起動

```cmd
npm run dev
```

### Step 3: ブラウザでハードリロード

1. ブラウザで `http://localhost:3000/survey` にアクセス
2. **Ctrl + Shift + R** を押してハードリロード

---

## それでもエラーが続く場合

ファイルを再度確認してください：

1. VS Codeで `lib/normalizeAnswers.ts` を開く
2. **176行目**を確認
3. `const singleNum = ...` になっているか確認
4. なっていない場合は、**Ctrl + S** で保存
